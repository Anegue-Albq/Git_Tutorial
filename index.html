Passos para Hospedar um Projeto no GitHub Pages

  1. Crie um repostório no Github com o mesmo nome do seu projeto
  2. No terminal, inicialize o repostório local com git init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o commit git commit -m "first commit"
  5. Conecte ao repostório remoto com:
    git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin main
